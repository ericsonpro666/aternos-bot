
# Bot de encendido automático Aternos + Discord

Este bot se conecta automáticamente a Aternos cada 5 minutos para revisar si tu servidor está apagado y encenderlo. Envía un mensaje al canal de Discord cuando lo hace.

## ¿Cómo usarlo en Railway?

1. Subí estos archivos a un nuevo repositorio en GitHub.
2. En Railway, crea un nuevo proyecto usando "Deploy from GitHub".
3. Configura las variables de entorno:

- USER
- PASSWORD
- DISCORD_TOKEN
- CHANNEL_ID

4. ¡Hecho! El bot se iniciará solo y se mantendrá activo 24/7.
